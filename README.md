## Overview

The To-Do List project is a demonstration of my web development skills, providing a practical and user-friendly application for managing daily tasks. This project includes a variety of features designed to enhance productivity and task organization.

## Features

- **Task Management**: Add, edit, and delete tasks effortlessly.
- **Task Categorization**: Organize tasks into different categories or lists.
- **Completion Tracking**: Mark tasks as completed to keep track of your progress.
- **Responsive Design**: Optimized for both desktop and mobile devices, ensuring a seamless user experience across different screen sizes.
- **Persistent Storage**: Tasks are saved locally, so you won't lose your list even if you close the browser.

## Technologies Used

- **HTML5**: Structuring the content of the application.
- **CSS3**: Styling the application for an attractive and responsive user interface.
- **JavaScript**: Implementing the functionality and interactivity of the application.
- **Webpack**: Bundling the application for optimized performance and ease of development.
- **ES6 Modules**: Organizing the codebase into modular and maintainable components.
- **LocalStorage**: Storing the task data persistently in the user's browser.

## Getting Started

### Prerequisites

To run this project locally, you will need to have Node.js and npm (or yarn) installed on your machine.

### Installation

1. **Clone the repository**:

    ```sh
    git clone https://github.com/ayuboketch/To-Do_list.git
    cd To-Do_list
    ```

2. **Install the dependencies**:

    ```sh
    npm install
    ```
    
    or

    ```sh
    yarn install
    ```

### Usage

To start the development server and run the project locally, use:

```sh
npm start
```

or

```sh
yarn start
```

To build the project for production, use:

```sh
npm run build
```

or

```sh
yarn build
```

## Deployment

The project is deployed using GitHub Pages. To deploy the latest version, ensure that the `dist` directory is up to date and push it to the `gh-pages` branch:

```sh
git subtree push --prefix dist origin gh-pages
```
